<h3>Descripción</h3>

Un addon de subtítulos para Stremio que proporciona subtítulos de LegendasDivx.

<h3>Configuración</h3>

Este addon requiere configuración ya que necesitas proporcionar tu token de acceso de [LegendasDivx](https://legendasdivx.pt). Necesitas registrarte en el sitio web y obtener tu token.
