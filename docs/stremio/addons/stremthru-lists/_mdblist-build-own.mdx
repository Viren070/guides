1. Create/log in to your [MDBList](https://mdblist.com/) account.
2. (Optional) Link Trakt if you want Trakt-synced workflows.

![mdblist login](/img/stremio/mdblist_login.png)

3. Open [MDBList](https://mdblist.com/) for movies, or [Shows](https://mdblist.com/tv/) for TV.

![mdblist movie search](/img/stremio/mdblist_movie_filters.png)
![mdblist tv search](/img/stremio/mdblist_tv_filters.png)

4. Configure your filters and click `Search`.
5. Use `Create List` and save your list.

![mdblist create list](/img/stremio/mdblist_create_trakt_list.png)

6. Open [My Lists](https://mdblist.com/mylists/) and copy the list URL/ID.
7. Paste it into StremThru Lists config and add it as a catalog.

:::tip
Free MDBList accounts have list limits. If you need more curated options quickly, use public lists from established curators.
:::
