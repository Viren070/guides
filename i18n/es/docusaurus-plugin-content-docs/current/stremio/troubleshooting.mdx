# Solución de Problemas

## No se encontraron streams

Si no estás viendo ningún stream, puede ser debido a varias razones:

### No tienes addons instalados

Asegúrate de tener addons instalados. Si no tienes ningún addon instalado, no verás ningún stream.

### Tus addons no están funcionando

Algunos addons pueden no estar funcionando. Intenta usar un addon diferente.

### El contenido no está disponible

El contenido que estás buscando puede no estar disponible en los addons que tienes instalados.

### Estás buscando contenido muy nuevo

El contenido muy nuevo puede no estar disponible inmediatamente. Intenta buscar contenido más antiguo.

### Estás buscando contenido muy oscuro

El contenido muy oscuro puede no estar disponible en los addons que tienes instalados.

## Stuttering/Buffering

Si tu stream está haciendo stuttering o buffering, puede ser debido a varias razones:

### Tu conexión a internet es demasiado lenta

Asegúrate de tener una conexión a internet rápida. Se recomienda una conexión de al menos 25 Mbps para contenido 4K.

### El servidor está sobrecargado

El servidor desde el que estás haciendo streaming puede estar sobrecargado. Intenta usar un stream diferente.

### Estás usando una VPN

Las VPN pueden ralentizar tu conexión a internet. Intenta desactivar tu VPN.

### Tu dispositivo es demasiado lento

Tu dispositivo puede ser demasiado lento para manejar el stream. Intenta usar un dispositivo más rápido.

## Pantalla roja en streams de debrid

Si estás viendo una pantalla roja al intentar reproducir un stream de debrid, puede ser debido a varias razones:

### Tu servicio debrid no tiene el archivo

Tu servicio debrid puede no tener el archivo que estás intentando reproducir. Intenta usar un stream diferente.

### Tu servicio debrid está caído

Tu servicio debrid puede estar caído. Verifica el estado de tu servicio debrid.

### Tu clave API es incorrecta

Tu clave API puede ser incorrecta. Verifica tu clave API en la configuración de tu addon.

### Tu cuenta de servicio debrid ha expirado

Tu cuenta de servicio debrid puede haber expirado. Verifica el estado de tu cuenta.

## Torrentio no está funcionando

Si Torrentio no está funcionando, puede ser debido a varias razones:

### Torrentio está caído

Torrentio puede estar caído. Verifica el estado de Torrentio en su [página de estado](https://torrentio.strem.fun/status).

### Tu configuración de Torrentio es incorrecta

Tu configuración de Torrentio puede ser incorrecta. Verifica tu configuración de Torrentio.

### Tu servicio debrid no está funcionando

Tu servicio debrid puede no estar funcionando. Verifica el estado de tu servicio debrid.

## El contenido aparece dividido

Si el contenido aparece dividido (por ejemplo, una película aparece como múltiples episodios), puede ser debido a varias razones:

### El torrent está dividido

El torrent puede estar dividido en múltiples archivos. Esto es normal para algunos torrents.

### El addon no está manejando el torrent correctamente

El addon puede no estar manejando el torrent correctamente. Intenta usar un addon diferente.

## El audio se reproduce pero solo pantalla negra

Si el audio se reproduce pero solo ves una pantalla negra, puede ser debido a varias razones:

### El códec de video no es compatible

El códec de video puede no ser compatible con tu dispositivo. Intenta usar un stream diferente.

### Tu dispositivo no puede manejar el video

Tu dispositivo puede no ser capaz de manejar el video. Intenta usar un dispositivo más potente.

### El archivo de video está corrupto

El archivo de video puede estar corrupto. Intenta usar un stream diferente.

## Un addon no está funcionando

Si un addon no está funcionando, puede ser debido a varias razones:

### El addon está caído

El addon puede estar caído. Verifica el estado del addon.

### Tu configuración del addon es incorrecta

Tu configuración del addon puede ser incorrecta. Verifica tu configuración del addon.

### El addon no es compatible con tu dispositivo

El addon puede no ser compatible con tu dispositivo. Intenta usar un addon diferente.

## Los subtítulos no aparecen correctamente

Si los subtítulos no aparecen correctamente, puede ser debido a varias razones:

### Los subtítulos no están disponibles

Los subtítulos pueden no estar disponibles para el contenido que estás viendo.

### Tu configuración de subtítulos es incorrecta

Tu configuración de subtítulos puede ser incorrecta. Verifica tu configuración de subtítulos en Stremio.

### El addon de subtítulos no está funcionando

El addon de subtítulos puede no estar funcionando. Intenta usar un addon de subtítulos diferente.
