### Descargar Stremio {#downloading-stremio}
Ve a la [página de descargas de Stremio](https://www.stremio.com/downloads) y descarga el paquete apropiado para tu sistema.

Si necesitas ayuda para determinar qué paquete descargar, consulta lo siguiente:

Para Windows, descarga lo siguiente:

![Windows Download](/img/stremio/win_download.png)

Para Mac OS, descarga uno de los siguientes dependiendo de tu versión de Mac OS:

![Mac Download](/img/stremio/mac_download.png)

Para Linux, descarga uno de los siguientes:
:::tip
Actualmente hay una versión beta de Stremio v5 para Linux disponible, pruébala ahora siguiendo esta
[publicación del blog de Stremio](https://blog.stremio.com/stremio-5-released-for-linux/)
:::
![Linux Download](/img/stremio/linux_download.png)

Hay paquetes disponibles para distribuciones populares de Linux como Debian y Arch Linux. Sin embargo, también hay un enlace de flatpak así como un enlace al código fuente.

El flatpak también se puede usar para instalar Stremio en el Steam Deck.

Una vez que hayas descargado el paquete para tu sistema, instálalo y abre Stremio.
Luego, inicia sesión con la cuenta que creaste anteriormente.
