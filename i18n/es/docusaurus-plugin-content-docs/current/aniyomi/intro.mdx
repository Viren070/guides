# Introducción

[Aniyomi](https://aniyomi.org/) es una aplicación de anime y manga para Android simple y fácil de usar. Es un fork de ~~Tachiyomi~~ [Mihon](https://mihon.app).

## Funciones

- Seguimiento con [MyAnimeList](https://myanimelist.net/), [AniList](https://anilist.co/), [Kitsu](https://kitsu.app/), [MangaUpdates](https://www.mangaupdates.com/), [Shikimori](https://shikimori.one/), [Simkl](https://simkl.com/) y [Bangumi](https://bgm.tv/)
- Descarga de manga y anime para visualización sin conexión
- Lectura de manga y visualización de anime desde varias fuentes
- Lector personalizable con múltiples visores y temas
