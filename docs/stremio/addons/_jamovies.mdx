JaMovies is an addon that provides streams from VidSrc

To install JaMovies either: 
1. Use this direct install link for [Stremio Web](https://web.stremio.com/#/addons?addon=https://eja-addon.vercel.app/manifest.json) or [Stremio App](stremio://eja-addon.vercel.app/manifest.json)
2. Head to the [addon configuration page](https://eja-addon.vercel.app/) and click `Install`
