Puedes hacer clic en los botones `Browse popular lists` o `Browse trending lists` para explorar listas populares y en tendencia.
También puedes usar el cuadro de búsqueda para encontrar listas.

Por ejemplo, puedes buscar `marvel`.
![trakt addon search lists example](/img/stremio/trakt_addon_search_lists_example.png)

Cuando encuentres una lista, haz clic en el botón `Add list`.

Repite este proceso para cualquier otra lista que quieras añadir.
