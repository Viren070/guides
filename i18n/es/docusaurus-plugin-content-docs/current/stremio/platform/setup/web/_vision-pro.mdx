Para obtener la mejor experiencia en Apple Vision Pro, deberías usar el Moon VR Player en Stremio Web.

Sigue [esta publicación del blog](https://blog.stremio.com/how-to-use-stremio-web-on-vision-pro/) para configurar Stremio Web en tu Apple Vision Pro.
Asegúrate de seleccionar el Moon VR Player como el reproductor externo al configurar Stremio Web.

Después de configurar Stremio Web, deberías asegurarte de que Moon VR Player esté instalado en tu Apple Vision Pro.

Limitaciones: - La transmisión de torrents no es posible. Esto significa que **se requiere un servicio debrid** para transmitir torrents. - Sin seguimiento del progreso de episodios. - Los subtítulos proporcionados por addons no se pueden usar (aún puedes usar los subtítulos incrustados proporcionados por el archivo de video).

Para resolver estas limitaciones, puedes [configurar un servidor de streaming en tu teléfono android o escritorio.](#servidor-stremio)
