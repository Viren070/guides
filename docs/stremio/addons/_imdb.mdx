This addon will add a catalogue that fetches movies and shows from IMDB. 
This addon requires no configuration, so it can be installed from directly within the Stremio app. 

IMDB Catalogs can optionally use an RPDB API key to fetch ratings for the movies and shows and display them on the posters.
To do this, you will need to [obtain an RPDB API key](faq#how-do-i-obtain-a-rpdb-api-key)

If you want to use the RPDB API key, you can do the following:
1. Head to the [IMDB Catalog Configuration page](https://1fe84bc728af-imdb-catalogs.baby-beamup.club/configure)
2. Enter your RPDB API key
3. Click Install


Otherwise, you can install the addon directly using the following links for:
- [Stremio App](stremio://1fe84bc728af-imdb-catalogs.baby-beamup.club/manifest.json) 
- [Stremio Web](https://web.stremio.com/#/addons?addon=https://1fe84bc728af-imdb-catalogs.baby-beamup.club/manifest.json)
