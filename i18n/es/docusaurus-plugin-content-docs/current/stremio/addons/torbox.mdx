<h3>Descripción</h3>

El addon Torbox es un addon desarrollado específicamente para Torbox (un servicio debrid).
No funcionará con otros servicios debrid (como Real-Debrid, Premiumize, etc).

<h3>Configuración</h3>

Para instalar el addon Torbox:
