# Forks

## ¿Qué es un fork?

Un fork de Aniyomi es una versión alternativa de la aplicación desarrollada por distintas personas.
Puede incluir funciones adicionales, un diseño diferente u otros cambios que la hacen única.

## Lista de forks

- [Animetail2](https://github.com/Dark25/Animetail2)
  - Envío a dispositivos (casting)
  - Discord RPC
  - Soporte para streaming de torrents
- [Animiru](https://aniyomi.org/forks/Animiru/)
  - Elimina funciones de Manga, enfocándose en Anime
  - Discord RPC
- [Kuukiyomi](https://aniyomi.org/forks/Kuukiyomi/)

  - Soporte para streaming de torrents
  - Discord RPC
