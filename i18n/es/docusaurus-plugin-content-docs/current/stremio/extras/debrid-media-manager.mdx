# Debrid Media Manager

[https://debridmediamanager.com/](https://debridmediamanager.com/)

Esto es sólo para quienes pagan por un servicio Debrid.
Sólo soporta Real Debrid y AllDebrid y tiene soporte parcial para TorBox.

### ¿Cómo usarlo?

Este sitio actúa como interfaz de tu servicio de debrid. Puedes buscar cualquier película o serie en él.

Si tienes una película o serie demasiado nicho como para tener torrents en caché, puedes buscarla haciendo clic en el botón de búsqueda.

Una vez en la página de la película o serie, puedes explorar los torrents disponibles.
También puedes hacer clic en `show x uncached`. Esto mostrará torrents que no están en caché.

Para torrents en caché, verás `Instant RD/AD`.
Puedes ver el torrent directamente desde Debrid Media Manager haciendo clic en `look inside` y luego `Watch`.

Para torrents sin caché, verás `DL with RD/AD`.
Al hacer clic, enviarás una solicitud a tu servicio de debrid para que empiece a descargar ese torrent.

Si quieres verlo en Stremio, asegúrate de que está en caché y añadido a tu biblioteca (haz clic en Instant RD/AD para añadirlo a tu biblioteca).
Para contenido sin caché, tendrás que esperar a que termine la descarga.

Luego, puedes:

- Usar el catálogo de debrid de la extensión Torrentio (debes tener esta opción activada en la configuración de Torrentio)
- Usar [Debrid Search](/stremio/addons/debrid-search), que mostrará el resultado cuando hagas clic en la película o serie que quieres ver.
  :::note
  Aunque Debrid Search proporciona un catálogo de tu biblioteca de debrid, aparecerá el error `The addon providing this content has been removed` si no tienes activa la opción de catálogo de debrid en la configuración de Torrentio.
  :::
- Usar el [DMM Cast Stremio Addon](/stremio/addons/dmm-cast)
