# Pimp My Stremio

:::warning
Pimp My Stremio (PMS) no se ha actualizado desde 2020.
:::

Pimp My Stremio es un gestor local de addons para Stremio. Los addons disponibles a través de PMS tienen capacidades superiores a los addons normales.
Puedes encontrar una lista de addons [aquí](https://github.com/sungshon/PimpMyStremio/blob/master/src/addonsList.json).

Lee más al respecto en su [GitHub](https://github.com/sungshon/PimpMyStremio). Allí encontrarás una guía de usuario.

Aun cuando no existe un paquete para Android, es posible usar Termux para hacerlo funcionar. Aquí tienes una [guía](https://gist.github.com/sleeyax/e9635eb352a4fcdf94194f763d743689) que explica cómo hacerlo.
