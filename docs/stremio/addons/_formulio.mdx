[Formulio](https://github.com/TheRaceDirector/formuliodev/) is a Stremio addon that focuses on Formula One content. 

You can find the announcement for this addon and more information on [this reddit post](https://www.reddit.com/r/FormulaStremio/comments/1e753gq/formulio_addon_relaunch_guide/)

Formulio can be [selfhosted](https://github.com/TheRaceDirector/formuliodev/). 

There is a public instance of Formulio available on ElfHosted [here](https://formulio.elfhosted.com/). 
:::info
For some reason, the `Install` button on the configuration page does not work. You can use the direct install links below to install Formulio.
:::

Since Formulio requires no configuration, you can simply click one of these direct install links below: 

- [Stremio App](stremio://formulio.elfhosted.com/manifest.json)
- [Stremio Web](https://web.stremio.com/#/addons?addon=https://formulio.elfhosted.com/manifest.json)
