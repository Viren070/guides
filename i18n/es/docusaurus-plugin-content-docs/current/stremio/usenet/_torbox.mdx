Para configurar una cuenta de Torbox con soporte para Usenet, sigue estos pasos:

1. Ve al [sitio web de Torbox](https://torbox.app/)
2. Haz clic en `Get Started For Free Now` o `LOGIN` en la esquina superior derecha para crear una cuenta.
3. Haz clic en `Don't have an account yet?` en la parte inferior de la página.
4. Completa los campos de email y contraseña y haz clic en `Continue`
5. Se te enviará un email para verificar tu dirección de correo. Haz clic en el enlace del email para verificar tu dirección de correo.
6. Luego serás llevado al Dashboard.
7. Ve a la [página de Suscripción](https://torbox.app/subscription?referral=9ca21adb-dbcb-4fb0-9195-412a5f3519bc).
   :::info
   Puedes recibir una alerta que dice `Do you want to use this referral code? If you already have a code, this will replace it.`
   Esto es porque el enlace anterior tiene un código de referido. Obtendrás 7 días gratis si aceptas mi código de referido.

   También puedes ingresar este código de referido manualmente en la parte inferior de la página:

   ```
   9ca21adb-dbcb-4fb0-9195-412a5f3519bc
   ```

   :::

8. Para poder acceder a Usenet, necesitas comprar el paquete de nivel más alto - el paquete `Pro`.
9. Haz clic en `Subscribe` en el paquete `Pro`.
10. Luego, sigue los pasos en la página de pago para completar tu compra.
