<h3>Descripción</h3>

Un addon de Stremio que proporciona subtítulos usando la API de [Wyzie Subs](https://sub.wyzie.ru/).

<h3>Configuración</h3>

Configurar este addon es muy simple.

1. Haz clic en el botón `Configure`.
2. Selecciona los idiomas para los que quieres subtítulos.
3. Luego haz clic en `Install in Stremio` o si prefieres [instalarlo manualmente](/stremio/faq#how-do-i-install-an-addon-manually), haz clic en `Copy URL to Clipboard`
