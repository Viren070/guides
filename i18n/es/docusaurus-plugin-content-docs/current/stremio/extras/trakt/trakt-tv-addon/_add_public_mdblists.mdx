Para listas de otros usuarios que encontraste en MDBList, vuelve a la pestaña donde tienes abierta la lista en MDBList, haz clic derecho sobre el icono de Trakt y selecciona `Copy link address`. Esto copiará la URL de la lista a tu portapapeles.
![mdblist trakt url](/img/stremio/mdblist_trakt_url_list.png)

Una vez tengas la URL, pégala en el cuadro `Paste list URL` y haz clic en `Add List`.
![trakt addon paste url](/img/stremio/trakt_addon_paste_url.png)

Repite este proceso para cualquier otra lista que quieras añadir.
