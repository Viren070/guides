<h3>Descripción</h3>

Un addon de Stremio que proporciona subtítulos de SubDL. Requiere una clave API

<h3>Configuración</h3>

Para usar este addon, necesitas registrarte para obtener una cuenta y obtener una clave API. Para hacer esto, sigue estos pasos:

1. [Regístrate para obtener una cuenta](https://subdl.com/panel/register) o [inicia sesión](https://subdl.com/panel/login?page=/panel/api) si ya tienes una cuenta.
2. Una vez que hayas iniciado sesión, ve a la [página de clave API](https://subdl.com/panel/api) y haz clic en el botón `Copy API Code` para copiar tu clave API.
3. En la página de configuración del addon, pega tu clave API en el campo `API Key`.
4. Si quieres tener subtítulos para idiomas distintos al inglés, puedes seleccionarlos del menú desplegable `Select Languages`.
5. Ahora que has configurado el addon, haz clic en el botón `Install` o `Install (Web)` para agregar el addon a tu aplicación Stremio.
