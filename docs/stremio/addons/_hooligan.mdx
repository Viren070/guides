Hooligan is an addon that provides live streams for sports events. 

You can look at the addons [announcement post](https://www.reddit.com/r/StremioAddons/comments/1f0vjc8/new_addon_hooligan_v0015_beta_formula1_soccer_mlb/) on r/StremioAddons for more information.

To install it, you can use one of the following links:

- [Addon Configuration Page](https://docker-hooligan.onrender.com/)

- [Stremio App Direct Install Link](stremio://docker-hooligan.onrender.com/manifest.json)

- [Stremio Web Direct Install Link](https://web.stremio.com/#/addons?addon=https://docker-hooligan.onrender.com/manifest.json)
