# Open in Stremio

Esta es una extensión de navegador que busca mejorar la integración entre Stremio y otros sitios web.

Tiene varias funciones:

- Añade un botón que abre la película o serie en Stremio en sitios como IMDb, Trakt, Letterboxd, Rotten Tomatoes y Metacritic.
- Permite buscar el texto resaltado en cualquier página en Stremio.

También puedes crear listas usando la extensión a partir de:

- publicaciones de reddit (en r/MovieSuggestions, r/televisionsuggestions, r/Letterboxd).
- resultados de búsqueda de IMDB.
- películas de BestSimilar.

Luego puedes usar [su herramienta que convierte listas en catálogos](https://www.journey.co.il/stremio/) para que estas listas aparezcan en Stremio.

Puedes encontrar capturas de pantalla y más información sobre cómo usar estas funciones en:

- [La página de GitHub de la extensión](https://github.com/yortem/chrome-stremio-imdb)
- [Publicación en Reddit anunciando la extensión](https://www.reddit.com/r/StremioAddons/comments/1dtvjmk/chrome_extension_open_the_movie_in_stremio/)
- [Publicación en Reddit que muestra cómo crear un catálogo basado en publicaciones de reddit](https://www.reddit.com/r/StremioAddons/comments/1ei94yp/create_a_catalog_based_on_reddit_suggestions/)
