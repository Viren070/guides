# Stremio Addons

<Tabs queryString="addon" className="hidden-tabs">
<TabItem default value="">
This page contains all the instructions for each addon covered in the [setup](setup#setting-up-addons) section.

To view the instructions for a specific addon, append `?addon=addon-name` to the URL, where `addon-name` is the name of the addon you want to view.

For example, to view the instructions for Streaming Catalogs, you would visit the following URL: 

https://guides.viren070.me/stremio/addons?addon=streaming-catalogs
