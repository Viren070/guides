Stremio se puede instalar desde la [Play Store](https://play.google.com/store/apps/details?id=com.stremio.one).

Sin embargo, también puedes descargar el APK para tu dispositivo desde la [página de descargas de Stremio](https://www.stremio.com/downloads)

:::warning
Stremio puede no estar disponible en tu región en la Play Store.
Sin embargo, puedes hacer sideload usando el APK del [sitio web de Stremio](https://www.stremio.com/downloads).

Puedes usar la aplicación [Downloader by AFTVnews](https://play.google.com/store/apps/details?id=com.esaba.downloader) en tu TV o la aplicación [Send Files to TV](https://play.google.com/store/apps/details?id=com.yablio.sendfilestotv) en tu teléfono para hacer sideload del APK.
:::

![Android TV Download](/img/stremio/android_tv_download.png)

Muy probablemente necesitarás descargar el APK ARM.

- ARM (armeabi-v7a) - 32 bit
- ARM64 (arm64-v8a) - 64 bit
- x86 - 32 bit Intel
- x86_64 - 64 bit Intel

Una vez que la aplicación Stremio esté instalada, ábrela e inicia sesión con la cuenta que creaste anteriormente.
