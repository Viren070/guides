Puedes usar un servidor independiente con el Servicio Stremio o usar el servidor que está integrado en la aplicación de escritorio.
Si planeas usar solo Stremio Web, entonces recomiendo usar el servidor independiente.

Para comenzar, primero asegúrate de que uno de los dos esté descargado e instalado.

1.  Descarga el [Servicio Stremio](https://www.stremio.com/download-service) independiente o la aplicación [Stremio Desktop](https://www.stremio.com/download).
2.  Ejecuta la configuración y luego inicia la aplicación.
    :::note
    Si estás usando el Servicio Stremio, puedes ver si está ejecutándose verificando la bandeja del sistema.
    :::
3.  Abre [Stremio Web](https://web.stremio.com/)
4.  Ve a Configuración
5.  Haz clic en la sección Streaming, deberías ver esto:

        ![configuración del servicio stremio](/img/stremio/service_settings.png)

    :::tip
    Si muestra el Estado como error, intenta deshabilitar cualquier bloqueador de anuncios/rastreadores que tengas habilitado.
    En Brave, no funcionará sin deshabilitar Brave Shields.

    También puedes necesitar agregar una excepción para el servicio Stremio en la configuración de tu firewall.
    :::

6.  El servicio Stremio ahora ha sido configurado correctamente. Sin embargo, solo puedes acceder a él en el mismo dispositivo.
    Para acceder desde otro dispositivo (en la misma red) sigue los siguientes pasos:

7.  Establece el endpoint HTTPS de Streaming a tu dirección IP local (esta opción debería aparecer como un menú desplegable).
8.  Luego copia la URL Remota y pégala en la configuración de Stremio de tu otro dispositivo para la URL del servidor de streaming.

El servicio Stremio ahora está configurado y se puede acceder desde otro dispositivo.
