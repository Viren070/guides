# Stremio Downloader

[Stremio Downloader](https://github.com/BurningSands70/stremio-downloader) es una aplicación de escritorio que permite descargar a tu dispositivo los streams que estás viendo en Stremio.

También puedes descargar un vídeo haciendo clic en los 3 puntos dentro del reproductor y luego en `Download this video`.

Si quieres descargar streams en Android, lee este [post de reddit](https://www.reddit.com/r/StremioAddons/comments/ekwj5x/how_to_download_moviesseries_on_android/).
