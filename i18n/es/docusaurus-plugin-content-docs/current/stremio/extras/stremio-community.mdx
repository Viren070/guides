# Stremio Community

[Stremio Community](https://github.com/Zaarrg/stremio-community-v5) es una versión comunitaria de la app de escritorio de Stremio para Windows con funciones adicionales.

Ofrece las siguientes características:

- **Discord Rich Presence**: Comparte en Discord lo que estás viendo.
- **Reproductor MPV totalmente personalizable**: Usa el reproductor MPV con configuraciones personalizadas, incluidos upscalers de anime integrados.
- **Soporte de extensiones del navegador**: Instala extensiones del navegador como addons.
- **Compatibilidad con modo oscuro**: Soporte para el modo oscuro de Windows.

y más.

Lee más sobre cómo usar sus funciones adicionales en la [página de GitHub](https://github.com/Zaarrg/stremio-community-v5).
