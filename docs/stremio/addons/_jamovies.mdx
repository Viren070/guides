[JaMovies](https://github.com/ejatapibeda/JaMovies/) is an addon that provides streams from various different providers.

The [public instance](https://eja-addon.vercel.app/) is currently down, however, you can self-host it by following the instructions on the [GitHub repository](https://github.com/ejatapibeda/JaMovies?tab=readme-ov-file#self-hosting-on-vercel)

If the public instance is back up, you can install it by clicking the following links:
- [Stremio App](stremio://eja-addon.vercel.app/manifest.json)
- [Stremio Web](https://web.stremio.com/#/addons?addon=https://eja-addon.vercel.app/manifest.json)
