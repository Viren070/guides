Para MDBList (sólo tus listas) y Couchmoney, haz clic en `Browse personal lists (requires login)`.
![trakt addon browse personal lists](/img/stremio/trakt_addon_browse_personal_lists.png)

    Luego haz clic en `Add list` en todas las listas que quieras añadir.

:::tip
Puedes usar el [MDBList Stremio Addon](https://1fe84bc728af-stremio-mdblist.baby-beamup.club/configure) para añadir listas que no estén sincronizadas con Trakt.
Es útil si alcanzaste el límite de listas en Trakt y quieres añadir más listas desde MDBList.
:::
