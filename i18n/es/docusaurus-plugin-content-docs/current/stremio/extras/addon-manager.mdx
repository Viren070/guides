# Administrador de Addons

El [Administrador de Addons original](https://addon-manager.dontwanttos.top/) permitía reordenar tus addons y, por lo tanto, sus catálogos.
Esto significa que ya no necesitas desinstalar y reinstalar addons para cambiar el orden de sus catálogos.

Luego, Redd-ravenn hizo un fork y añadió la capacidad de renombrar tus addons y renombrar o eliminar catálogos específicos de addons.
Su fork está disponible en GitHub: [Redd-ravenn/stremio-addon-manager](https://github.com/Redd-ravenn/stremio-addon-manager).

No hay una instancia pública “oficial” de esta versión, pero puedes desplegarla tú mismo con Docker o Node.js siguiendo las instrucciones del README del repositorio.
Recomiendo usar servicios como Vercel o Netlify para alojarla. Haz un fork del repositorio y despliega en uno de estos servicios.

Sin embargo, he configurado una instancia pública del fork de Redd-ravenn en https://addon-manager.viren070.me/ ([Código fuente](https://github.com/Viren070/stremio-addon-manager)) para quienes no quieran desplegarla por su cuenta.
