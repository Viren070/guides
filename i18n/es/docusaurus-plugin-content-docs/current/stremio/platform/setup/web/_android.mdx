Si estás usando Stremio Web en tu teléfono android, entonces puedes agregarlo a tu pantalla de inicio para fácil acceso. Aquí están los pasos para hacer esto:

1. **Abre Stremio Web**: Abre tu navegador y ve a [Stremio Web](https://web.stremio.com/).
2. **Agrégalo a tu pantalla de inicio**:
   - Los pasos para esto son diferentes dependiendo del navegador que estés usando.
   - Sin embargo, generalmente, puedes hacer esto tocando los tres puntos y seleccionando "Agregar a pantalla de inicio."
3. **Accede a la aplicación**: El ícono de Stremio aparecerá en tu pantalla de inicio, permitiéndote acceder a Stremio Web directamente.

Aún no hemos terminado. Stremio Web, por sí solo, no soporta reproducción de torrents o streaming de videos en el navegador.

Para superar estas limitaciones, puedes [configurar un servidor de streaming en tu teléfono android o escritorio.](#servidor-stremio)

Otro método para usar Stremio Web en tu teléfono android es usar un reproductor externo. Este método, sin embargo, tiene algunas limitaciones:

    - La transmisión de torrents aún no es posible. Esto significa que **se requiere un servicio debrid** para transmitir torrents.
    - Sin seguimiento del progreso de episodios.
    - Los subtítulos proporcionados por addons no se pueden usar (aún puedes usar los subtítulos incrustados proporcionados por el archivo de video).

Aquí están los pasos para configurar un reproductor externo:

1. **Abre la Aplicación Stremio Web**: Toca el ícono de Stremio en tu pantalla de inicio para abrir la aplicación.
2. **Descartar la Advertencia**: Ignora el banner "servidor de streaming no disponible" tocando `No mostrar de nuevo`.
3. **Acceder a Configuración**:
   - Ve al menú de configuración tocando el ícono de engranaje.
   - Desplázate hacia abajo a la sección `Reproductor` y bajo `Avanzado` selecciona `Reproducir en reproductor externo`
4. **Elige un Reproductor Externo**: Selecciona `VLC`, `MX Player` o `Just Player`. Asegúrate de que el reproductor elegido esté instalado en tu dispositivo.

Ahora puedes abrir un enlace y debería abrirse en el reproductor externo que seleccionaste.
